Project Title: Spam vs Ham Email Classifier using NLP & Machine Learning

Overview: This project builds a machine learning model to classify emails/messages as Spam or Ham (Not Spam) using Natural Language Processing techniques.

The workflow includes: Text preprocessing, feature extraction using TF-IDF

Tech Stack involved: Python, Scikit-learn, Pandas, NumPy, Matplotlib, NLP preprocessing (stopwords, tokenization, TF-IDF)

ML Pipeline: Data Cleaning, Text Preprocessing, TF-IDF Vectorization, Model Training, Matrix Evaluation
